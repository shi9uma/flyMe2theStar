# csapp

深入理解计算机系统，[Computer Systems: A Programmer's Perspective](https://csapp.cs.cmu.edu/)，一个比较全面综合的课程，涵盖了计算机组成、操作系统、汇编、C 语言以及 Linux 系统编程

这里放在学习和阅读本书时一些记录、思考以及感悟等，内容包含书本、lab、视频学习以及其他参考资料等

主要以 **chapter-title.md** 的方式来编排
